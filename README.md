# Al-Furqan-QUIZ
Interactive Quran App
Al Furqan Quiz · v8.3

Un jeu interactif pour tester et améliorer la connaissance du Coran à travers des versets authentiques.


---

Objectif

Le but du jeu est simple :

Identifier de quelle sourate provient un verset.

Le jeu utilise des versets réels et propose différents niveaux de difficulté et modes de jeu.


---

Modes de jeu

1. Mode QCM

Un verset est affiché

4 propositions de sourates

Le joueur doit choisir la bonne réponse


Récompense

Bonne réponse : +1 point

Mauvaise réponse : affichage de la bonne sourate



---

2. Mode Devinette

Un verset est affiché

Le joueur doit trouver la sourate sans propositions

Sélection parmi toutes les sourates


Système d’indices (progressif)

À chaque erreur, des indices sont donnés :

Direction (avant / après dans le Coran)

Distance (nombre de sourates)

Juz

Type (mecquoise / médinoise)

Nombre de versets



---

Difficulté

Chaque verset est classé :

Facile

Moyen

Difficile



---

Fonctionnalités principales

Multilingue : FR / AR / EN

Thèmes visuels (Gold, Claire, Sombre)

Versets authentiques

Score en temps réel

Écran de fin avec performance



---

Système de masquage

Certains mots du verset peuvent être masqués automatiquement pour augmenter la difficulté.

Objectif

Empêcher de reconnaître immédiatement la sourate.

Fonctionnement

Le jeu utilise un fichier de données contenant les mots à masquer

Les mots sont remplacés par un masque visuel

Le système gère les variations arabes, les suffixes et les harakats



---

Intelligence artificielle

Le jeu intègre un système intelligent permettant :

La sélection optimisée des versets

L’adaptation de la difficulté

L’amélioration progressive de l’expérience utilisateur



---

Source des données

Les versets sont extraits d’un fichier contenant :

Les noms de sourates

Les versets correspondants

Les mots à masquer



---

Lancer le jeu

1. Ouvrir le fichier HTML


2. Choisir la taille de la série


3. Sélectionner le mode de jeu


4. Cliquer sur Commencer
